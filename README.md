

## Description

vpn-api

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# watch mode
$ pnpm start:watch

# development
$ pnpm build
$ pnpm start


# production mode
$ pnpm build
$ pnpm start:prod
```

## Test

```bash
# unit tests
$ pnpm test

# e2e tests
$ pnpm test:e2e

# test coverage
$ pnpm test:cov
```
